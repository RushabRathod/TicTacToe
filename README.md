# TicTacToe Game

TicTacToe is a classic two-player game built using HTML, CSS, and JavaScript. It provides a simple yet engaging user interface where two players can compete against each other in a game of TicTacToe.

## Features

- *Interactive Gameplay*: Users can click on the grid to make their moves, with alternating turns between 'X' and 'O'.
- *Winning Detection*: The game automatically detects and highlights winning combinations.
- *Reset Functionality*: Players can reset the game board to start a new game at any time.
- *Responsive Design*: Fully responsive user interface, ensuring a seamless experience across devices.

## Technologies Used

- *Frontend*: HTML, CSS, JavaScript

## Setup Instructions

1. Clone the repository: https://github.com/YourUsername/TicTacToe.git

2. Open the index.html file in your web browser to start playing the game.


## Usage

1. *Game Start*: Open the index.html file in a web browser.

2. *Making Moves*: Click on any cell in the TicTacToe grid to place your marker ('X' or 'O').
3. *Winning* : The game will automatically highlight the winning combination and display the winner.
4. *Resetting the Game*:  Click the "Reset" button to clear the board and start a new game.
